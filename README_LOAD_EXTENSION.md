How to load FocusMesh MVP in Chrome:

1) Open Chrome -> chrome://extensions/
2) Enable Developer mode (toggle top-right)
3) Click "Load unpacked" and select this folder: focusmesh-extension
4) The extension should appear in the toolbar (puzzle icon -> pin it)
5) To debug:
   - Open a webpage, right-click -> Inspect -> Console to see content script output.
   - In chrome://extensions/ find FocusMesh and click "Service worker" to view background logs.
